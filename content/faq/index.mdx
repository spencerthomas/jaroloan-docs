# Frequently Asked Questions

Find answers to common questions about using JaroLoan.

## Getting Started

### How do I log in?

1. Navigate to your organization's JaroLoan URL
2. Enter your email address
3. Enter your password
4. Click **Sign In**

### How do I reset my password?

1. Click "Forgot Password" on the login screen
2. Enter your email address
3. Check your email for reset link
4. Follow link to create new password

### What browsers are supported?

JaroLoan supports modern browsers:

- Chrome 90+
- Firefox 88+
- Safari 14+
- Edge 90+

### How do I contact support?

- Email: support@jaroloan.com
- In-app: Click the help icon (?)
- Documentation: You're here!

## Loans

### How do I create a new loan?

1. Click **+ New Loan** in the pipeline
2. Select loan type
3. Enter loan parameters
4. Add borrower information
5. Save the loan

See [Create First Loan Tutorial](/docs/tutorials/create-first-loan)

### How do I find a specific loan?

Use the search bar at the top of the pipeline:

- Search by borrower name
- Search by loan number
- Search by property address

### Can I delete a loan?

Loans cannot be permanently deleted for compliance reasons. You can:

- Archive inactive loans
- Mark as withdrawn
- Mark as denied

### How do I change the loan officer on a loan?

1. Open the loan
2. Click **Assign** or the LO name
3. Select new loan officer
4. Confirm the change

## Documents

### What file types are supported?

Supported formats:

- PDF (recommended)
- Images (JPG, PNG)
- Microsoft Office (DOC, DOCX, XLS, XLSX)

### What's the maximum file size?

Maximum file size is 25MB per file.

### How do I request documents from borrowers?

1. Navigate to **Documents** tab
2. Click **Request Documents**
3. Select document types needed
4. Add instructions
5. Send request

### Where are documents stored?

Documents are stored securely in encrypted cloud storage with:

- AES-256 encryption
- Geographic redundancy
- 7-year retention

## Disclosures

### When is the Loan Estimate due?

The LE must be delivered within **3 business days** of receiving a complete application.

### What triggers a re-disclosure?

Re-disclosure required for:

- Changed circumstances
- Borrower-requested changes
- Rate lock events
- Settlement delays

See [TRID Compliance](/docs/compliance/trid)

### How do I know if the CD is compliant?

JaroLoan automatically:

- Calculates tolerance
- Compares to LE
- Flags violations
- Calculates cures

### Can I waive the 3-day waiting period?

In limited circumstances with:

- Documented bona fide personal financial emergency
- Written waiver from borrower
- Not available for all situations

## Conditions

### What's the difference between PTD and PTF?

| Type | Name | Cleared By |
|------|------|------------|
| PTD | Prior to Docs | Before CD generation |
| PTF | Prior to Funding | Before funding |
| PTC | Prior to Close | Before closing |

### Who can waive conditions?

Condition waiver requires:

- Appropriate role (typically UW or Manager)
- Documented justification
- May require manager approval

### How do I track condition status?

View condition status in:

- Loan **Conditions** tab
- Dashboard widgets
- Pipeline indicators
