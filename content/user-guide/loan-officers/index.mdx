# Loan Officer Guide

This guide covers everything Loan Officers need to know about using JaroLoan LOS effectively.

## Your Dashboard

The LO Dashboard provides:

- **Pipeline Overview** - Your active loans by stage
- **Lead Queue** - Incoming leads for follow-up
- **Tasks** - Action items requiring attention
- **Performance Metrics** - Pull-through rate, volume, speed

## Creating Loans

### From Lead

1. Select lead from queue
2. Click **Convert to Loan**
3. Add additional information
4. Loan created in Application stage

### New Loan

1. Click **New Loan**
2. Select loan type
3. Enter loan parameters
4. Add borrower(s)
5. Save loan

## Managing Your Pipeline

### Pipeline View

Access your pipeline:

- Click **Pipeline** in sidebar
- Default view shows your loans
- Filter by stage, date, status

### Loan Status

Monitor loan health:

- **On Track** - Progressing normally
- **At Risk** - SLA concern
- **Stale** - No activity

## Working with Borrowers

### Adding Borrowers

1. Open loan detail
2. Navigate to **Borrowers** tab
3. Click **Add Borrower**
4. Complete borrower form

### Co-Borrowers

Add co-borrowers when applicable:

- Same process as primary
- Link to primary borrower
- Specify relationship

### Communication

Track all borrower communication:

- Internal notes
- Emails (logged)
- Phone calls (manual entry)

## Disclosures

### Generating LE

1. Navigate to **Disclosures** tab
2. Click **Generate Loan Estimate**
3. Review figures
4. Send to borrower

### Capturing Intent

After LE delivered:

1. Wait for borrower acknowledgment
2. Record Intent to Proceed
3. Document method and date

## Pricing and Lock

### Running Scenarios

Compare pricing options:

1. Click **Scenarios**
2. Adjust parameters
3. View pricing comparison
4. Select best option

### Rate Lock

Lock in rate:

1. Click **Lock Rate**
2. Select lock period
3. Confirm lock
4. Monitor expiration

## Handoff to Processing

When ready for processing:

1. Complete application
2. Verify all basic info
3. Click **Submit to Processing**
4. Processor assigned

## Best Practices

1. **Respond quickly** - Lead response time matters
2. **Complete applications** - Reduces back-and-forth
3. **Set expectations** - Communicate timelines
4. **Monitor SLAs** - Stay on track
