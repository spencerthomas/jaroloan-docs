# Processor Guide

This guide covers everything Processors need to know about efficiently processing loans in JaroLoan.

## Your Dashboard

The Processor Dashboard shows:

- **Task Queue** - Items requiring your attention
- **Assigned Loans** - Loans you're processing
- **SLA Tracker** - Time-sensitive items
- **Pending Verifications** - Awaiting results

## Receiving Loans

### Assignment

Loans assigned via:

- Automatic round-robin
- Manual assignment by manager
- Self-assignment from queue

### Initial Review

When receiving a new loan:

1. Review loan summary
2. Check completeness
3. Identify missing items
4. Create task list

## Document Collection

### Requesting Documents

1. Identify needed documents
2. Navigate to **Documents** tab
3. Click **Request**
4. Select document types
5. Send to borrower

### Portal Uploads

When borrower uploads:

1. Receive notification
2. Review document
3. Classify if needed
4. Link to conditions

### Manual Collection

For documents received outside portal:

1. Upload document
2. Select type
3. Add source notes
4. Link to conditions

## Ordering Verifications

### Credit

1. Navigate to **Verifications** tab
2. Click **Order Credit**
3. Confirm borrower consent
4. Submit order

### VOI (Income)

1. Click **Order VOI**
2. Select provider
3. Verify employer info
4. Submit order

### VOE (Employment)

1. Click **Order VOE**
2. Enter employer details
3. Select contact method
4. Submit order

### VOA (Assets)

1. Click **Order VOA**
2. Have borrower authenticate
3. Retrieve account data
4. Review results

## Preparing for Underwriting

### Submission Checklist

Before submitting:

- [ ] All income documented
- [ ] Employment verified
- [ ] Assets verified
- [ ] Property info complete
- [ ] Credit reviewed

### Submit to UW

1. Verify checklist complete
2. Click **Submit to Underwriting**
3. Add submission notes
4. Confirm submission

## Managing Conditions

### Post-UW Conditions

After underwriting decision:

1. Review conditions list
2. Identify document needs
3. Request from borrower
4. Track responses

### Condition Clearing

When documents received:

1. Review document
2. Verify meets condition
3. Link document to condition
4. Route for UW review

## Communication

### Borrower Contact

Log all communication:

- Notes in loan timeline
- Email tracking
- Phone call summaries

### LO Communication

Keep LO informed:

- Status updates
- Borrower issues
- Timeline concerns

## Best Practices

1. **Process in order** - First in, first out
2. **Touch it once** - Complete tasks fully
3. **Document everything** - Add notes
4. **Communicate proactively** - No surprises
