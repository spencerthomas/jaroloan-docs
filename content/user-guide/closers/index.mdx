# Closer Guide

This guide covers the closing process from Clear to Close through funding in JaroLoan.

## Your Dashboard

The Closer Dashboard displays:

- **Closing Calendar** - Scheduled closings
- **Funding Queue** - Ready to fund
- **CD Status** - Disclosure tracking
- **Post-Close** - Pending completion

## Clear to Close

### Receiving CTC Loans

When loan reaches CTC:

1. Receive notification
2. Review final conditions
3. Verify all cleared
4. Begin closing prep

### Final Review

Before CD generation:

- [ ] All conditions cleared
- [ ] Final approval in place
- [ ] Insurance confirmed
- [ ] Title cleared

## Closing Disclosure

### Generating CD

1. Navigate to **Disclosures** tab
2. Click **Generate CD**
3. Review all figures:
   - Loan terms
   - Projected payments
   - Costs at closing
   - Cash to close
4. Run LE/CD comparison
5. Verify tolerance compliance

### LE/CD Comparison

System compares automatically:

- Zero tolerance items
- 10% tolerance bucket
- Unlimited items
- Flags violations

### Sending CD

1. Review final CD
2. Click **Send to Borrower**
3. Select delivery method
4. Confirm sending
5. 3-day period begins

### 3-Day Waiting Period

Track waiting period:

- System calculates dates
- Shows earliest closing date
- Alerts for timing issues

## CD Revisions

### When to Revise

**New 3-day wait required:**
- APR increase > 0.125% (fixed)
- APR increase > 0.25% (variable)
- Loan product change
- Prepayment penalty added

**No new wait:**
- Minor corrections
- Seller credit changes
- Clerical fixes

### Creating Revision

1. Click **Revise CD**
2. Make necessary changes
3. Document reason
4. Generate revised CD
5. Send to borrower

## Scheduling Closing

### Coordinate with Parties

- Settlement agent
- Borrower availability
- Wire timing
- Recording deadlines

### Set Closing Date

1. Navigate to **Closing** tab
2. Select closing date/time
3. Enter location
4. Save schedule

## Closing Day

### Final Figures

Confirm day-of figures:

1. Verify final numbers
2. Confirm wire amount
3. Update if changes

### Documents

Track signing status:

- Loan package sent
- Signatures received
- Notarization complete

## Funding

### Pre-Funding Checklist

Before funding:

- [ ] All documents signed
- [ ] PTF conditions cleared
- [ ] Wire instructions verified
- [ ] Figures balanced

### Authorize Funding

1. Navigate to **Funding** tab
2. Review checklist
3. Click **Authorize Funding**
4. Enter wire confirmation

### Recording

After funding:

1. Track recording status
2. Enter recording info
3. Mark loan as Funded

## Post-Close

### Final Documents

Collect remaining items:

- Signed note
- Recorded deed
- Final title policy
- Insurance binder

### Investor Delivery

Prepare investor package:

1. Generate delivery checklist
2. Compile documents
3. Submit to investor

## Best Practices

1. **Start early** - Begin CD prep at CTC
2. **Communicate timing** - Set expectations
3. **Double-check figures** - Accuracy matters
4. **Track wires** - Confirm receipt
