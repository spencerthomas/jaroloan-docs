# Changelog

All notable changes to JaroLoan are documented here.

## Version Format

We use [Semantic Versioning](https://semver.org/):

- **Major**: Breaking changes
- **Minor**: New features (backward compatible)
- **Patch**: Bug fixes

---

## v1.0.0 - Initial Release

*Released: January 2024*

### Features

- **Complete Loan Lifecycle** - Full support from lead to post-close
- **TRID Compliance** - Automated LE/CD with tolerance tracking
- **Condition Management** - Create, track, and clear conditions
- **Document Management** - Upload, classify, and organize documents
- **Verification Integration** - Credit, VOI, VOE, VOA
- **AUS Integration** - DU and LP submission
- **Multi-Role Dashboard** - Role-specific interfaces
- **Borrower Portal** - Self-service for borrowers
- **Audit Trail** - Complete activity logging
- **HMDA Support** - Data collection and reporting

### Technical

- Next.js 15 with App Router
- PostgreSQL database
- Prisma ORM
- shadcn/ui components
- Real-time WebSocket updates
