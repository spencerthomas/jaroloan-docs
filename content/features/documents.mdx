# Document Management

JaroLoan provides comprehensive document management for mortgage files.

## Document Overview

The Documents section provides:

- Centralized file storage
- Auto-classification
- Version control
- Expiration tracking
- Secure access

## Uploading Documents

### Drag and Drop

1. Navigate to **Documents** tab
2. Drag files to upload area
3. Documents auto-classify
4. Review and confirm types

### Browse Upload

1. Click **Upload**
2. Select files
3. Choose document type
4. Add notes (optional)
5. Click **Save**

### Multi-File Upload

Upload multiple files simultaneously:

- Select multiple files
- Each classified separately
- Review all before saving

## Auto-Classification

### How It Works

JaroLoan uses AI to classify documents:

1. OCR extracts text
2. ML model analyzes content
3. Document type suggested
4. Confidence score provided

### Classification Accuracy

High confidence (>90%): Auto-accepted
Medium confidence: Review recommended
Low confidence: Manual classification required

### Manual Override

If classification incorrect:

1. Open document
2. Click **Reclassify**
3. Select correct type
4. Save changes

### Categories

### Search

## Version Control

### Multiple Versions

Each document maintains version history:

- Original preserved
- New versions tracked
- Compare versions
- Restore if needed

### Viewing History

1. Open document
2. Click **Version History**
3. View all versions
4. Select to view or restore
