# Verifications

JaroLoan integrates with verification providers for automated income, employment, and asset verification.

## Verification Types

### Credit Reports

Tri-merge credit reports from bureaus:

- Experian
- Equifax
- TransUnion

### VOI (Verification of Income)

Automated income verification:

- IRS tax transcripts
- Payroll data
- Bank income analysis

### VOE (Verification of Employment)

Employment history verification:

- Current employment
- Previous employers
- Self-employment

### VOA (Verification of Assets)

Asset account verification:

- Bank accounts
- Investment accounts
- Retirement accounts

## Ordering Verifications

### Credit Report

1. Navigate to loan **Verifications** tab
2. Click **Order Credit**
3. Confirm borrower info
4. Submit request
5. Results return in seconds

### VOI

1. Click **Order VOI**
2. Select provider
3. Review borrower consent
4. Submit request
5. Results typically 1-3 days

### VOE

1. Click **Order VOE**
2. Enter employer information
3. Select verification method
4. Submit request

### VOA

1. Click **Order VOA**
2. Borrower authenticates accounts
3. Account data retrieved
4. Review results

## Reviewing Results

### Credit Review

Review credit data:

- Credit scores (all bureaus)
- Trade lines
- Public records
- Inquiries
- Fraud alerts

### Income Review

Review income findings:

- W-2 income
- 1099 income
- Tax return data
- YTD income

### Employment Review

Review employment data:

- Current employer confirmation
- Start date
- Position/title
- Income

### Asset Review

Review asset findings:

- Account balances
- Average balance
- Large deposits
- Transaction history

## Verification Expiration

Verifications have validity periods:

| Type | Validity |
|------|----------|
| Credit | 120 days |
| VOI | 120 days |
| VOE | 10 days of closing |
| VOA | 60-90 days |

### Refresh Process

When verification expires:

1. System flags expiration
2. Click **Refresh**
3. New verification ordered
4. Updated results reviewed

## Manual Entry

For verifications not available electronically:

1. Click **Manual Entry**
2. Enter verification data
3. Upload supporting documentation
4. Mark as manually verified

<Callout type="info">
Manual verifications require supporting documentation and may have additional review requirements.
</Callout>

## Fraud Detection

Verification services include fraud monitoring:

- SSN validation
- Identity verification
- Address verification
- Employment fraud flags
