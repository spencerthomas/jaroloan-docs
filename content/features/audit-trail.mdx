# Audit Trail

JaroLoan maintains a complete audit trail of all actions for compliance and accountability.

## What's Tracked

### User Actions

All user actions are recorded:

- Logins and logouts
- Data changes
- Document access
- Decisions made

### System Actions

Automated actions tracked:

- Notifications sent
- Status changes
- Scheduled tasks
- API calls

### Data Changes

Field-level audit history:

- Previous value
- New value
- Changed by
- Timestamp

## Viewing Audit History

### Loan Timeline

The Timeline tab shows:

- Chronological activity
- Action type icons
- User attribution
- Timestamps

### Field History

View specific field changes:

1. Hover over field
2. Click history icon
3. View all changes

### Document Access Log

Track document views:

- Who viewed
- When
- Download history

## Audit Reports

### Activity Report

Generate reports showing:

- User activity summary
- Date range filter
- Action type filter

### Compliance Report

For regulatory requirements:

- Disclosure delivery
- Decision history
- Document chain of custody

### Export Options

Export audit data:

- CSV format
- PDF report
- Date range selection

## Retention

### Data Retention Policy

Audit logs retained per compliance requirements:

| Data Type | Retention |
|-----------|-----------|
| Loan activity | 7 years |
| User logins | 2 years |
| Document access | 7 years |
| System logs | 1 year |

### Archival

After retention period:

- Archived to cold storage
- Retrievable upon request
- Compliant deletion

## Compliance Uses

### Regulatory Exams

Audit trail supports:

- CFPB examinations
- State audits
- Investor audits

### Dispute Resolution

Use audit trail to:

- Verify actions taken
- Confirm timelines
- Document communications

### Fair Lending

Track and demonstrate:

- Consistent decision-making
- Non-discriminatory practices
- Equal treatment

## Security

### Tamper-Proof

Audit logs are:

- Immutable
- Time-stamped
- Hash-verified
- Separately stored

### Access Control

Audit log access:

- Read-only for most users
- Admin can view all
- Cannot modify entries
