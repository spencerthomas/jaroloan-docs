# Pipeline Management

The Pipeline is your primary workspace for viewing and managing loans in JaroLoan.

## Pipeline Overview

The pipeline displays all loans in a unified view with:

- Stage-based organization
- Key loan metrics
- Quick actions
- Status indicators

## Accessing the Pipeline

1. Log in to JaroLoan
2. Click **Pipeline** in the left sidebar
3. Pipeline loads with default filters

## Pipeline Views

### Table View

Default view showing loans in rows:

- Sortable columns
- Customizable visible columns
- Inline status indicators

### Kanban View

Visual board organized by stage:

- Drag-and-drop capability
- Stage column summaries
- Card-based loan display

## Filtering

### Quick Filters

One-click filters for common views:

- My Loans
- All Active
- By Stage
- Stale Loans

### Advanced Filters

Combine multiple criteria:

- Loan Officer
- Processor
- Stage range
- Date range
- Loan amount
- Loan type

### Saved Filters

Save frequently used filter combinations:

1. Set up desired filters
2. Click **Save Filter**
3. Name your filter
4. Access from saved filters list

## Sorting

Sort by any visible column:

- Click column header to sort
- Click again to reverse
- Multi-column sort available

## Search

### Quick Search

Search bar at top of pipeline:

- Borrower name
- Loan number
- Property address
- Email

### Advanced Search

Access from search icon:

- Boolean operators
- Field-specific search
- Date ranges

## Bulk Actions

Select multiple loans for batch operations:

1. Check boxes next to loans
2. Click **Bulk Actions**
3. Select operation
4. Confirm action

Available bulk actions:

- Assign to user
- Change stage
- Export selected
- Send notifications

## Pipeline Statistics

Dashboard shows key metrics:

- Total loans by stage
- Pipeline volume
- Conversion rates
- Average time-in-stage

## Permissions

Pipeline access varies by role:

| Role | Sees |
|------|------|
| Loan Officer | Own loans |
| Processor | Assigned loans |
| Underwriter | In UW queue |
| Manager | All team loans |
| Admin | All loans |

## Tips & Best Practices

- Use saved filters for your daily workflow
- Monitor stale loans regularly
- Review SLA indicators
- Use keyboard shortcuts (press `?`)
