# Tutorial: Create Your First Loan

This tutorial walks you through creating your first loan in JaroLoan.

**Time:** 10 minutes
**Level:** Beginner
**Prerequisites:** Active JaroLoan account

## What You'll Learn

- How to start a new loan
- Entering basic loan information
- Adding borrower details
- Saving and viewing your loan

## Step 1: Access New Loan

1. Log in to JaroLoan
2. Click **Pipeline** in the left sidebar
3. Click the **+ New Loan** button in the top right

<Callout type="tip">
Keyboard shortcut: Press `N` to start a new loan from anywhere.
</Callout>

## Step 2: Select Loan Type

Choose the loan purpose:

- **Purchase** - Buying a property
- **Refinance - Rate/Term** - Refinancing for better terms
- **Refinance - Cash-Out** - Refinancing to access equity

For this tutorial, select **Purchase**.

## Step 3: Enter Loan Information

Fill in the basic loan details:

### Loan Amount

Enter the requested loan amount:

- Input: `350000`
- This is the amount to be borrowed

### Property Type

Select the property type:

- Single Family
- Condo
- Multi-Family (2-4 units)
- Townhouse

Select **Single Family**.

### Occupancy

Choose how the property will be used:

- Primary Residence
- Second Home
- Investment Property

Select **Primary Residence**.

### Purchase Price

For purchase loans, enter the purchase price:

- Input: `400000`

## Step 4: Click Continue

After entering loan information:

1. Review your entries
2. Click **Continue**
3. System validates the information

## Step 5: Add Primary Borrower

Now add the borrower information:

### Name

- First Name: `John`
- Last Name: `Smith`

### Contact Information

- Email: `john.smith@email.com`
- Phone: `555-123-4567`

### Additional Details (Optional)

You can add more details now or later:

- Date of Birth
- SSN
- Current Address

## Step 6: Save Borrower

1. Review borrower information
2. Click **Save Borrower**
3. Borrower is added to the loan

## Step 7: Add Property Address

Enter the property being purchased:

### Address

- Street: `123 Main Street`
- City: `Springfield`
- State: Select your state
- ZIP: `12345`

### Property Details

- Year Built: `2010`
- Square Feet: `2000`

## Step 8: Save Property

1. Review property information
2. Click **Save Property**
3. Property is added to the loan

## Step 9: View Your Loan

After saving, you're on the Loan Detail page:

### Overview Tab

Shows key loan metrics:

- Loan amount
- Property value
- LTV ratio
- Stage: Application

### Borrower Tab

Shows borrower information:

- Name and contact
- Edit capability

### Conditions Tab

Currently empty (no conditions yet)

### Documents Tab

Ready for uploads

### Timeline Tab

Shows creation activity
