# Tutorial: Process a Loan to Close

This comprehensive tutorial walks through the entire loan process from application to funding.

**Time:** 30 minutes
**Level:** Intermediate
**Prerequisites:** Understanding of [Create First Loan](/docs/tutorials/create-first-loan)

## Overview

This tutorial covers:

1. Application completion
2. Disclosure delivery
3. Processing and verifications
4. Underwriting submission
5. Condition clearing
6. Closing and funding

## Part 1: Application Stage

### Complete Application

Ensure all application data is entered:

1. Open loan detail page
2. Navigate to **Borrower** tab
3. Complete all required fields:
   - Personal information
   - Employment history
   - Income details
   - Asset information

### Generate Loan Estimate

1. Navigate to **Disclosures** tab
2. Click **Generate Loan Estimate**
3. Review all figures
4. Click **Send to Borrower**

<Callout type="warning">
LE must be sent within 3 business days of application.
</Callout>

### Capture Intent to Proceed

After borrower reviews LE:

1. Receive borrower intent
2. Navigate to **Disclosures** tab
3. Click **Record Intent**
4. Select method (verbal/written/electronic)
5. Enter date and details
6. Save

## Part 2: Processing Stage

### Submit to Processing

1. Verify application complete
2. Click **Submit to Processing**
3. Processor assigned

### Order Verifications

As Processor:

1. Navigate to **Verifications** tab
2. Order Credit Report:
   - Click **Order Credit**
   - Confirm borrower info
   - Submit
3. Order VOI:
   - Click **Order VOI**
   - Select provider
   - Submit
4. Order VOE:
   - Click **Order VOE**
   - Enter employer
   - Submit

### Collect Documents

1. Navigate to **Documents** tab
2. Click **Request Documents**
3. Select required items:
   - Paystubs (most recent)
   - W-2s (last 2 years)
   - Bank statements (2 months)
   - ID
4. Send request to borrower

### Monitor Document Uploads

As borrower uploads:

1. Receive notification
2. Review uploaded documents
3. Classify if needed
4. Request additional if needed

## Part 3: Underwriting

### Submit to Underwriting

When processing complete:

1. Verify all verifications received
2. Review submission checklist
3. Click **Submit to Underwriting**
4. Add notes if needed

### Underwriting Review

Underwriter reviews file:

1. Receives loan in queue
2. Reviews credit, income, assets
3. Analyzes property
4. Makes decision

### Receive Decision

Decision options:

- **Approved** - No conditions
- **Conditional Approval** - Conditions required
- **Suspended** - More info needed
- **Denied** - Does not qualify

For this tutorial, assume **Conditional Approval**.

## Part 4: Condition Clearing

### View Conditions

1. Navigate to **Conditions** tab
2. Review all conditions:
   - PTD (Prior to Docs)
   - PTF (Prior to Funding)

### Clear Conditions

For each condition:

1. Request document if needed
2. Receive document from borrower
3. Link document to condition
4. Review document
5. Click **Clear** or route to UW

### Final Approval

When all PTD conditions cleared:

1. Underwriter reviews
2. Issues final approval
3. Loan moves to Clear to Close

## Part 5: Closing

### Generate Closing Disclosure

1. Navigate to **Disclosures** tab
2. Click **Generate CD**
3. Review all figures
4. Verify LE/CD comparison
5. Check tolerance compliance
6. Click **Send to Borrower**

<Callout type="info">
3-day waiting period begins when borrower receives CD.
</Callout>

### Schedule Closing

1. Coordinate with all parties
2. Navigate to **Closing** tab
3. Set closing date/time
4. Confirm location

### Closing Day

1. Confirm final figures
2. Borrower signs documents
3. Collect signed documents

## Part 6: Funding

### Clear PTF Conditions

Before funding:

1. Review PTF conditions
2. Ensure all cleared
3. Verify all requirements met

### Authorize Funding

1. Navigate to **Funding** tab
2. Review checklist
3. Verify wire instructions
4. Click **Authorize Funding**
5. Enter wire confirmation

### Mark as Funded

1. Confirm wire sent
2. Click **Mark Funded**
3. Loan moves to Funded stage

## Completion

Congratulations! You've processed a loan from application to funding.

### Post-Close Tasks

- Collect final documents
- Prepare investor delivery
- Archive loan file

## Summary

You learned how to:

- Complete the application process
- Generate and send disclosures
- Order verifications
- Submit for underwriting
- Clear conditions
- Generate CD and close
- Fund the loan
