# Quick Start Guide

This guide will help you get started with JaroLoan LOS. By the end, you'll have created your first loan application.

## Prerequisites

Before you begin, ensure you have:

- An active JaroLoan account with login credentials
- A modern web browser (Chrome, Firefox, Safari, or Edge)
- Basic understanding of mortgage loan terminology

## Step 1: Log In

1. Navigate to your JaroLoan instance URL
2. Enter your email address and password
3. Click **Sign In**

<Callout type="info">
If you've forgotten your password, click "Forgot Password" on the login screen to receive a reset link via email.
</Callout>

## Step 2: Explore the Dashboard

After logging in, you'll see the **Pipeline Dashboard**:

- **Left Sidebar** - Navigation to different sections
- **Main Area** - Your loan pipeline with stage filters
- **Quick Actions** - Create new loans, search, etc.

## Step 3: Create Your First Loan

1. Click the **New Loan** button in the top-right corner
2. Select the loan type (Purchase, Refinance, etc.)
3. Fill in the basic information:
   - Loan amount
   - Property type
   - Occupancy
4. Click **Continue**

## Step 4: Add Borrower Information

1. Enter the primary borrower's details:
   - First and last name
   - Email address
   - Phone number
2. Add additional information as needed
3. Click **Save Borrower**

## Step 5: Add Property Details

1. Enter the property address
2. Select property type and occupancy
3. For purchases, enter the sale price
4. Click **Save Property**

## Step 6: View Your Loan

After saving, you'll see the **Loan Detail** page with:

- **Overview** - Key loan metrics and status
- **Borrower** - All borrower information
- **Conditions** - Required items (empty for new loans)
- **Documents** - Uploaded files
- **Timeline** - Activity history

## Next Steps

- [Order Verifications](/docs/features/verifications) - Pull credit, VOI, VOE
- [Manage Conditions](/docs/tutorials/manage-conditions) - Track underwriting requirements
