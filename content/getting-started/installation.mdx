# Installation & System Requirements

This guide covers the system requirements and setup process for JaroLoan LOS.

## System Requirements

### Browser Requirements

JaroLoan LOS is a web-based application that runs in modern browsers:

| Browser | Minimum Version |
|---------|-----------------|
| Chrome | 90+ |
| Firefox | 88+ |
| Safari | 14+ |
| Edge | 90+ |

### Network Requirements

- Stable internet connection (minimum 5 Mbps recommended)
- WebSocket support for real-time updates
- HTTPS enabled (all connections are encrypted)

### Display

- Minimum resolution: 1280 x 720
- Recommended: 1920 x 1080 or higher
- Responsive design supports tablets (landscape mode)

## Account Setup

### For Organizations

Contact JaroLoan sales to set up your organization:

1. **Organization provisioning** - We create your isolated environment
2. **Admin account creation** - You receive admin credentials
3. **User onboarding** - Admin can create user accounts

### For Individual Users

Your administrator will provide:

- Login URL for your organization's instance
- Username (typically your email)
- Temporary password (change on first login)

## First Login

1. Navigate to your organization's JaroLoan URL
2. Enter your credentials
3. Complete password change if prompted
4. Set up two-factor authentication (if required)
5. Review and accept terms of service

## Browser Configuration

For optimal performance:

- Enable JavaScript
- Allow cookies for the JaroLoan domain
- Enable notifications (for alerts)
- Disable pop-up blockers for the JaroLoan domain

## Next Steps

Once logged in, continue to the [Quick Start Guide](/docs/getting-started/quick-start) to begin using the system.
