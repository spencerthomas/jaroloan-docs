# Introduction to JaroLoan LOS

JaroLoan LOS is a modern, full-featured mortgage loan origination system designed to streamline the entire loan lifecycle from application to funding.

## What is JaroLoan?

JaroLoan is a cloud-based loan origination system (LOS) that provides:

- **Complete Loan Lifecycle Management** - From lead capture through post-closing
- **TRID-Compliant Disclosures** - Automated LE/CD generation with tolerance tracking
- **Multi-Role Workflow** - Purpose-built interfaces for LOs, processors, UWs, and closers
- **Automated Verifications** - Integrated credit, income, employment, and asset verification
- **Comprehensive Audit Trail** - Full history of every action for compliance

## Key Features

### Pipeline Management

Track all loans in a unified pipeline view with:

- Stage-based organization
- Customizable filtering and sorting
- SLA tracking and alerts
- Bulk actions for efficiency

### Condition Tracking

Manage underwriting conditions with:

- Prior-to-docs and prior-to-funding categorization
- Document linking and request workflows
- Automated clearing based on document classification

### TRID Compliance

Built-in compliance features including:

- 3-business-day tracking for LE delivery
- Intent to Proceed capture and documentation
- Tolerance bucket monitoring
- CD 3-day waiting period management

### Borrower Portal

Self-service portal for borrowers to:

- Upload required documents
- Review and sign disclosures
- Track loan status
- Communicate with their loan team

## User Roles

JaroLoan supports multiple user roles with specific permissions:

| Role | Primary Functions |
|------|-------------------|
| **Loan Officer** | Lead management, loan creation, borrower communication |
| **Processor** | Document collection, verification ordering, file prep |
| **Underwriter** | Risk analysis, decision making, condition creation |
| **Closer** | CD generation, closing coordination, funding |
| **Admin** | User management, system configuration, reporting |

## Technology Stack

JaroLoan is built on modern technologies:

- **Frontend**: Next.js 15 with React and TypeScript
- **UI Components**: shadcn/ui with Tailwind CSS
- **Database**: PostgreSQL with Prisma ORM
- **Authentication**: NextAuth.js with role-based access
- **Real-time**: WebSocket support for live updates
