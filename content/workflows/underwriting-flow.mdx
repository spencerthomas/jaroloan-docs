# Underwriting Workflow

This document details the underwriting workflow in JaroLoan, from submission through decision.

## Submission to Underwriting

### Pre-Submission Checklist

Before submitting to underwriting:

- [ ] Credit report pulled and reviewed
- [ ] Income documentation complete
- [ ] Employment verified
- [ ] Assets verified
- [ ] Appraisal received (if applicable)
- [ ] All required disclosures sent

### Submitting a Loan

1. Navigate to the loan detail page
2. Click **Submit to Underwriting**
3. Confirm submission checklist
4. Loan enters underwriting queue

## Underwriting Review

### Queue Assignment

Loans are assigned based on:

- Underwriter workload
- Loan complexity
- Product type expertise
- Round-robin distribution

### Review Process

Underwriters follow a structured review:

1. **File Overview** - High-level review of loan parameters
2. **AUS Analysis** - Review DU/LP findings
3. **Credit Analysis** - Credit history, scores, explanations
4. **Income Analysis** - Stability, documentation, calculation
5. **Asset Analysis** - Sufficiency, sourcing, seasoning
6. **Property Analysis** - Value, condition, title

## Decision Types

### Approve

Loan meets all guidelines with no additional conditions.

### Approve with Conditions

Loan is approvable pending satisfaction of conditions:

- **Prior-to-Docs (PTD)** - Must clear before CD
- **Prior-to-Funding (PTF)** - Must clear before funding
- **Prior-to-Close (PTC)** - Must clear before closing

### Suspend

Loan cannot be decided without additional information:

- Material deficiencies identified
- Additional documentation required
- Requires re-review after documents received

### Deny

Loan does not meet guidelines:

- Adverse action notice required
- Specific denial reasons documented
- Appeal process available

## Condition Management

### Creating Conditions

Conditions can be created from:

- Templates (recommended)
- Manual entry
- AUS findings import

### Condition Best Practices

- Be specific about requirements
- Link to relevant guidelines
- Set appropriate due dates
- Include document type needed

## Re-Underwriting

Loans may require re-underwriting when:

- Significant loan changes occur
- New material information received
- Conditions reveal new issues
- Rate/program changes
